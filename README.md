🔍 PokéAPI Search App

Uma aplicação web simples e intuitiva que consome a PokéAPI para buscar e navegar por Pokémon em tempo real.

## 🚀 Funcionalidades

* **Busca Dinâmica:** Encontre Pokémon digitando o **Nome** ou o **ID**.
* **Navegação Rápida:** Botões de "Próximo" e "Anterior" para navegar diretamente pela Pokédex.
* **Responsivo:** Interface adaptada para diferentes tamanhos de tela.

## 🛠️ Tecnologias Utilizadas

* HTML5 / CSS3
* JavaScript (ES6+)
* [PokéAPI](https://pokeapi.co/) (Consumo de API REST)
